IN GAME : 

Controller Input :
Joystick Left: Character Movement
Joystick Right: Rotate Camera
X: Light Attack
A: Jump
B: Defense
Y: Change Weapon
LT: Special Attack
Back: Respawn when dead
Start: Pause

Keyboard Input :
WASD: Character Movement 
Mouse orientation: Rotate Camera
Left Click: Light Attack
Space: Jump
Q: Special Attack
E: Change Weapon
Right Click: Defense
Esc: Pause
